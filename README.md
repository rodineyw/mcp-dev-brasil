## What is this?

A collection of [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) servers for Brazilian services. Each server gives AI agents (Claude, ChatGPT, Copilot, etc.) the ability to interact with Brazilian APIs — payments, fiscal, banking, and more.

**No MCP servers existed for most Brazilian services.** This repo fills that gap.

## Quick Start

### Using with Claude Desktop

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "asaas": {
      "command": "npx",
      "args": ["-y", "@codespar/mcp-asaas"],
      "env": {
        "ASAAS_API_KEY": "your-api-key"
      }
    }
  }
}
```

### Using with any MCP client

```bash
npx @codespar/mcp-asaas
```

## Contributing

We welcome contributions! See [CONTRIBUTING.md](docs/CONTRIBUTING.md) for guidelines.

### Adding a new server

1. Create a directory under `packages/<category>/<service>/`
2. Implement the MCP server following the pattern of existing servers
3. Add tests
4. Submit a PR

### Requesting a server

Open an [issue](https://github.com/codespar/mcp-brasil/issues) with the "server request" label.
